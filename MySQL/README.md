# MySQL
MySQL是一种开源的关系型数据库管理系统（RDBMS），广泛用于存储、管理和检索结构化数据。当前大多数公司都会用到MySQL数据库，所以掌握MySQL数据库的使用是必须的。<br>

笔者使用的是Ubuntu 18.04.6系统，就以此讲解 MySQL 数据库的安装、配置信息查看和常规操作。<br>
